Introducción

Visual Studio Code (VS Code) es un editor de código gratuito que se puede usar para programar en muchos lenguajes, como JavaScript, Python, C++, entre otros.

Es liviano, rápido y se puede personalizar con extensiones, temas y configuraciones. Es ideal para principiantes y profesionales por igual.

Instalación

Podés descargar VS Code desde su página oficial. Está disponible para Windows, macOS y Linux.

Solo tenés que elegir tu sistema operativo y seguir los pasos del instalador.

Interfaz

La interfaz principal tiene varias partes importantes:

  • Explorador: donde ves tus archivos.
  • Editor: donde escribís tu código.
  • Terminal: para escribir comandos.
  • Barra lateral: para extensiones, Git, búsqueda, etc.

También podés dividir el editor en varias pestañas para ver varios archivos a la vez.

Abrir archivos

Podés abrir una carpeta completa desde Archivo > Abrir Carpeta. Esto es útil si trabajás en un proyecto.

También podés abrir archivos individuales si estás haciendo algo más simple.

Atajos
  • Ctrl + P: Buscar y abrir un archivo.
  • Ctrl + Shift + P: Paleta de comandos.
  • Ctrl + /: Comentar una línea.
  • Ctrl + ñ: Abrir o cerrar la terminal.
  • Ctrl + B: Ocultar o mostrar la barra lateral.

Podés cambiar los atajos desde la sección de configuración si querés.

Terminal

La terminal integrada es muy útil para correr comandos sin salir de VS Code.

Se abre con Ctrl + ñ y podés tener varias pestañas de terminal abiertas.

Extensiones

Las extensiones te ayudan a agregar funciones nuevas, como soporte para otros lenguajes o herramientas útiles.

Algunas populares:

  • Prettier: para formatear el código.
  • Live Server: para ver tu sitio web mientras lo editás.
  • ESLint: para detectar errores en JavaScript.

Podés buscar extensiones en la pestaña con el ícono de cuadrito.

Temas

Podés cambiar el aspecto de VS Code con temas de colores.

Hay temas claros, oscuros, y de muchos estilos distintos.

Se cambian desde Preferencias > Tema de Color.

Configuración

Desde Archivo > Preferencias > Configuración podés ajustar cómo se comporta el editor.

Por ejemplo:

{
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  "files.autoSave": "onWindowChange"
}

También podés usar la interfaz gráfica si no querés tocar el JSON directamente.

Control de versiones

VS Code tiene integración con Git para guardar el historial de cambios de tu proyecto.

  • Ver los cambios en los archivos.
  • Hacer commit y push.
  • Clonar repositorios desde GitHub.

Es muy útil si trabajás en equipo o querés tener un backup de tu código.

Snippets

Los snippets son atajos para escribir fragmentos de código más rápido.

Por ejemplo, podés escribir "html5" y que te aparezca toda la estructura base de HTML automáticamente.

Se pueden crear snippets propios desde Preferencias > Fragmentos de código.

Consejos finales

Empezar con VS Code puede parecer mucho, pero con el tiempo se vuelve muy cómodo.

No tengas miedo de probar cosas nuevas, buscar extensiones o ajustar configuraciones.

La comunidad es muy activa y hay muchos tutoriales gratis para seguir aprendiendo.
